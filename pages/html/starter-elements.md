The following elements should serve you well as an initial set to become familiar with:

* `head`
* `body`
* `title`
* `p`
* `a`
* `img`
* `h1`, `h2`, `h3`, `h4`, `h5`, `h6`
* `ul`, `ol`, `li`
* `blockquote`
* `br`, `hr`
* `em`, `strong`, `small`
* `nav`
* `article`
* `header`
* `footer`
* `section`
